xlxd reflector install script.

Can be used to install a new reflector, or re-install an existing reflector.
If re-installing (restoring) an existing reflector, please place the following 2 files in the /tmp folder BEFORE running the script:

config.inc.php AND callinghome.php

The script installs an Nginx webserver

xlxd dashboard files can be found in /var/www/html/dashboard


INSTALLATION

cd /root

git clone https://github.com/va3uv/xlxd-install.git
